## .:[ ConVars ]:.
```
// Enable Chat Command Silencer Plugin || 1= Yes || 0= No
cmd_slint_enable "1"

// Silent Command Chat Begin With (!) || 1= Yes || 0= No
cmd_slint_exclamation "1"

// Silent Command Chat Begin With (.) || 1= Yes || 0= No
cmd_slint_dot "0"

// Silent Command Chat Begin With (/) || 1= Yes || 0= No
cmd_slint_slash "0"
```
